Nmap, an open source application for Pentesting

Posted by hailsa on September 19, 2021

This tool is open source software and was created by Gordon Lyon to track and discover ports and try it. Nmap provides a number of features for probing computer networks, including host discovery and service and operating system detection. Below, all you can do with Nmap:

  • Server discovery: Identify computers on a network, for example, by listing those that respond to ping.
  • Identify open ports on a target computer.
  • Determine what services are running it.
  • Determine what operating system and version that computer uses, (this technique is also known as fingerprinting).
  • It has some characteristics of the network hardware of the machine under test.
